# ShareP2P

A playtime with webrtc using peerJs

## Installation

```bash
yarn install
```

## Getting Started

First, run the web server:

```bash
# run web server
npm run dev
# or
yarn dev
# or
pnpm dev
```

Second, run peerjs-server (signaling server)

```bash
# run peerjs server
npm run peer:server
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
